# finnhub_project
# 📈 Finnhub Stock Market Project

A Flask-based stock market web application using the Finnhub API.

## 🚀 Features

- Search stock companies by symbol
- View company information
- View live stock market data
- Fetch data from Finnhub API
- Store company data in MySQL
- JSON data handling
- Flask web UI

---

## 🛠️ Tech Stack

- Python
- Flask
- MySQL
- HTML/CSS
- Finnhub API

---

## 📊 Company Information

The application displays:

- Company Name
- Symbol
- Currency
- Exchange (MIC)
- FIGI
- ISIN
- Stock Type

---

## 📈 Live Stock Data

The application shows live:

- Current Price
- High Price
- Low Price
- Open Price
- Previous Close
- Change %
- Market Movement
